# Saving examples

DSNTK provides a set of examples ready to play with.

To install examples in e.g. **examples** directory, enter:

```shell
$ dsntk exs examples
```

The structure of example directories and files is shown below:

```shell
$ tree .
.
└── examples
    ├── e1
    │   ├── e1.ctx
    │   └── e1.feel
    ├── e2
    │   ├── e2.ctx
    │   └── e2.dmn
    └── e3
        ├── e3.ctx
        └── e3.dtb

5 directories, 6 files
```
