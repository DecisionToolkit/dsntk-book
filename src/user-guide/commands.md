# Command-line commands

## -h --help

Print this message or the help of the given subcommand(s):

```shell
$ dsntk -h
```

or

```shell
$ dsntk --help
```

## -V --version

Print version information.

```shell
$ dsntk -V
```

or

```shell
$ dsntk --version
```

## srv

Runs ^dsntk as a service. To display all available options of the **srv** command, enter:

```shell
$ dsntk help srv
```
